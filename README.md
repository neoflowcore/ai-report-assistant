# AI Report Assistant

## 프로젝트명

바이브 코딩으로 완성하는 보고서 혁신

## 프로젝트 개요

본 프로젝트는 사용자가 지역, 뉴스 링크, 부동산 이슈, 현장 조사 변동률, 과거 우수 보고서 형식을 입력하면 AI가 보고서 작성에 최적화된 프롬프트와 지역분석 보고서 초안을 생성하는 웹 기반 업무 어시스턴트입니다.

## 주요 기능

- 지역분석 보고서 입력 화면 제공
- 뉴스 링크 및 부동산 이슈 입력
- 현장 조사 변동률 입력
- 과거 우수 보고서 형식 참고
- 보고서 작성용 AI 프롬프트 자동 생성
- 보고서 초안 자동 출력

## 기술 스택

- HTML5
- Tailwind CSS
- JavaScript
- GitHub Pages
