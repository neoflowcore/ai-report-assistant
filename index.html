공공기관 내부 업무용 보고서 지원 도구

모두의 부동산 보고서

부동산 정책·지역 현황·수치 자료를 바탕으로 AI 보고서 작성용 프롬프트, 수정 프롬프트, 분량 조정 프롬프트를 생성합니다.

01

보고서 재료 입력

보고서 주제, 분석 지역, 수치 자료, 사례, 참고 보고서 내용을 입력하세요.

안내

사용 방법

아래 순서대로 진행하면 보고서 초안 작성 흐름이 정리됩니다.

  1. 보고서 재료 입력 왼쪽에 보고서 주제, 지역, 이슈, 수치, 사례, 참고 보고서 내용을 입력합니다.
  2. AI 프롬프트 생성 입력값을 바탕으로 보고서 초안 작성용 프롬프트를 생성합니다.
  3. 복사 후 AI 도구 열기 ChatGPT 또는 Gemini를 열고 복사된 프롬프트를 붙여넣습니다.
  4. 보고서 초안 보완 AI가 작성한 초안을 수정 영역에 붙여넣고 보완 지시문을 생성합니다.
  5. 분량 조정 보고서 초안을 붙여넣고 목표 분량을 입력해 글자 수 조정 프롬프트를 생성합니다.

입력한 내용은 서버나 데이터베이스에 저장되지 않고, 사용자의 브라우저 안에서만 처리됩니다.

03

AI 보고서 작성용 프롬프트

생성된 프롬프트를 복사해 AI 도구에 붙여넣으세요.

04

보고서 초안 수정 영역

AI가 작성한 초안을 붙여넣고 수정 요청을 생성하세요.

05

보고서 분량 조정

AI가 생성한 보고서 초안을 목표 글자 수나 비율에 맞게 조정하는 프롬프트를 생성합니다.
